Cl: 塩素

製法

酸化マンガン

MnO2(酸化マンガン(IV))に、濃塩酸を加えて、加熱して得られる。

  1. 上にあげた2つをまぜまぜして加熱
    以下反応式
    MnO2 + 4HCl → MnCl2 + 2H2O +Cl2
    半反応式で書いたら以下の通り
    • MnO2 + 4H+ + 2e- → 2H2O + Mn2+
    • 2Cl- → 2Cl2 + 2e-
  2. 水を使って、1.の反応で残ったHClを取り除く
  3. 濃H2SO4aqを使って、1.の反応で残ったH2Oを取り除く。

補足: 2.と 3. を入れ替えると、H2Oを取り除くことができず、純粋なCl2が得られない。

高度さらし粉

Ca(ClO)2 · 2H2O(高度さらし粉)に、塩酸を加えて得られる。

以下反応式
Ca(ClO)2 · 2H2O + 4HCl → CaCl2 + 4H2O + 2Cl2

半反応式で書いたら以下の通り
  • ClO- + 2H+ + e- → Cl- + H2O
  • 2Cl- → Cl2 + 2e-

高度さらし粉の語呂合わせ: "隠れる鬼に水"

さらし粉

CaCl(ClO) · H2O(さらし粉)に、塩酸を加えて得られる。

以下反応式
CaCl(ClO) · H2O + 2HCl → CaCl2 + 2H2O + Cl2

さらし粉の語呂合わせ: "ｶｧｧｧｧ(Ca), くるくるお水"

反応

with H2

H2とは, 光が当たるとき, 反応する[する/しない]。
光が当たらない冷暗所では, 反応しない[する/しない]。

反応して, HClを生じる。

with H2O

H2Oに溶かすことで, 一部が反応して, HCl と, HClOを生じる。

Cl2 + H2O ⇄ HCl + HClO

次亜塩素酸と仲間たち

化学式 名前
HClO 次亜塩素酸
HClO2 亜塩素酸
HClO3 塩素酸
HClO4 過塩素酸

Cl2 + H2O ⇄ HCl + HClO

次亜塩素酸は、酸化採用が強い。 以下酸化を示す半反応式。

ClO- + 2H+ +2e- → Cl- + H2O

Br: 臭素

臭化マグネシウム(MgBr2)を酸化させて得られる。

反応

with H2

加熱と触媒を用いた反応によって, HBrを生じる。

with H2O

少し溶ける

I: ヨウ素

ヨウ化カリウム(KI)を酸化させて得られる。

反応

with H2

加熱と触媒を用いた反応によって, HIを少し生じる。

with H2O

水に溶けにくい。

ヨウ素液

製法

I2は水に溶けにくい。

ただし、ヨウ化カリウム水溶液(KI aq)に入れることで、三ヨウ化イオン(I3-)を生じながら溶ける。

そして、液体の「ヨウ素ヨウ化カリウム溶液」というのができる。(これが一般に言うヨウ素液である。)

反応

ヨウ素液をデンプン水溶液とまぜまぜすると、青紫色になる。

これを、「ヨウ素デンプン反応」という。

ヨウ化カリウムデンプン紙

KIとでんぷんで構成された紙。

酸化剤の検出に使う。

ハロゲン化水素

ハロゲンと水素の化合物のこと。

ハロゲン化水素は、すべて無色で、刺激臭の有毒な気体である。

また、水によく溶ける。

HCl: 塩化水素

製法

塩化ナトリウムに, 濃硫酸を加えて加熱する。
発生した気体は下方置換法で集める。

NaCl + H2SO4 → NaHSO4 + HCl
(揮発性の酸の塩) + (不揮発性酸) → (不揮発性の酸の塩) + (揮発性の酸)

HF: フッ化水素

製法

蛍石(CaF2)に, 濃硫酸を加えて加熱する。

CaF2 + H2SO4 → CaSO4 + 2HF
(揮発性の酸の塩) + (不揮発性酸) → (不揮発性の酸の塩) + (揮発性の酸)

性質

HF aq はガラスの主成分であるSiO2を溶かし, ヘキサフルオロケイ酸(化学式: H2SiF6) を発生させる。

SiO2 + HF(aq) → H2SiF6 + 2H2O
