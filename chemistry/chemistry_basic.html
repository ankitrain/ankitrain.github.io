物質

元素と原子

  • 同素体
    同じ元素からなり、性質の異なる単体を互いに呼ぶ呼び方。
    S: 斜方硫黄, 単斜硫黄, ゴム状硫黄
    C: ダイヤモンド, 黒煙, フラーレン, カーボンナノチューブ
    O: 酸素分子, オゾン
    P: 赤リン, 黄リン
  • 同位体
    原子番号が同じで質量数が異なる原子を互いに呼ぶ呼び方。
    陽子の数が同じで中性子の数が異なる
    これらのうち、放射線を発するものを放射性同位体と呼ぶ
  • 壊変
    放射性同位体が、放射線を放射して別の原資に代わること。
    • α壊変
      Heの原子核が放出される壊変
    • β壊変
      e-が放出される壊変
    • γ壊変
      高いエネルギーを持つ電磁波が放出される壊変
  • アルカリ金属
    Hを除く1族
  • アルカリ土類金属
    Be, Mgを除く2族
  • ハロゲン
    F, Clなどの17族
  • 希ガス
    Arなどの18族。安定している。

イオン

  • 第1イオン化エネルギー
    原子から電子1個を取って、1価の陽イオンにするために必要なエネルギー
    アルカリ金属(1族)で最も小さく、希ガス(18族)で最も大きい。
    同族元素で比較した場合は、原子番号の大きいほうが小さい[大きい/小さい]。
  • 電子親和力
    原子から電子をもらって陰イオンになるときに放出されるエネルギー
    ハロゲン(17族)で大きい。

分子

  • 電気陰性度
    原子の共有電子対をひきつける強さ。

酸と塩基

酸塩基の定義

ブレンステッド・ローリーの定義
酸: H+を相手に与える物質
塩基: H+を相手から受け取る物質

電離

電離度
電離の割合のこと。
1に近いものは「強酸」や「強塩基」、否なものは「弱酸」、「弱塩基」
計算式
α = n' / n = c' / c
変数リスト
  • α: 電離度
  • n [mol]: 溶かした酸・塩基の量
  • n' [mol]: 電離した酸・塩基の量
  • c [mol/L]: 溶かした酸・塩基のモル濃度
  • c' [mol/L]: 電離した酸・塩基のモル濃度

水素イオン濃度

水素イオン指数(pH)
水素イオンの濃度を簡単に示すやつ
pH = -log10 [ H+ ]

[ H+ ] = 1.0 × 10-α [mol/L] のときに pH = α

量的関係

濃度が c [mol/L] の a [価] の酸の水溶液 v [L] と、 濃度が c' [mol/L] の a' [価] の塩基の水溶液 v' [L] が完全に中和したとき、

acv = a'c'v'

イオン化傾向

Li, K, Ca, Na, Mg, Al, Zn, Fe, Ni, Sn, Pb, H2, Cu, Hg, Ag, Pt, Au
リッチに貸そうかな、まだある、あてにすんな、酷すぎる借金

半反応式の作り方

  1. 左辺に反応前の物質、右辺に反応後の物質を並べる。(次の中見出しセクションで暗記できる)
    e.g.) MnO4- → Mn2+
  2. O原子の数をH2O を増やすことで、無理やりそろえる
    e.g.) MnO4- → Mn2+ + 4H2O
  3. H原子の数をH+を増やすことで、無理やりそろえる。
    e.g.) MnO4- + 8H+ → Mn2+ + 4H2O
  4. 電荷の数を、e-を増やすことで、無理やりそろえる。
    e.g.) MnO4- + 8H+ + 5e- → Mn2+ + 4H2O

酸化数

原子の参加の程度を示すもので、参加と還元を判断しやすくなる魔法の道具

下の取り決めに従い、酸化数が増加したときその物質は酸化されたといい、酸化数が減少したとき、還元されたという。

とりきめ

以下、例でXnMm: {X: a, Y: b}の形で示している場合、XnMmという物質においての、X原子の酸化数はaで、M原子の酸化数はbであるということとする。
  1. 単体中の原子の酸化数は0
    e.g.) H2: {Hの酸化数:0}, Na: {Na:0}
  2. 短原子イオンの原資の酸化数はイオンの電荷に等しい
    e.g.) Na+: {Na: +1}
  3. 化合物中のHの酸化数は+1, Oの酸化数は-2
    e.g.) H2O: {H:+1, O:-2}
  4. 化合物中の各原子の酸化数の総和は0である。
    e.g.) CO2: {C: +4, O: -2}
    上の例では (+4) + (-2) × 2 = 0 が成立する
  5. 多原子イオンの各原子の酸化数の総和は多原子イオンの電荷に等しい
    e.g.) SO42-: {S: +6 O: -2}
    上の例では (+6) + (-2) × 8 = -2 が成立する
