有機化合物

  1. 主に O と C と H で構成されていてめちゃくちゃ種類が多い
  2. 分子からなる物質で, 融点沸点は低い[高い/低い]
  3. 水に溶けにくく, 石油, アルコール, エーテルなど有機溶媒に溶けるものが多い.
  4. 燃焼しやすいものが多い

元素分析

  1. もともとの試料の質量を正確に測定する.

  2. 試料を乾燥した酸素中で完全燃焼させる.
    CO2, H2O が出てくるので, 塩化カルシウム管で H2O を, ソーダ石灰管で CO2 を吸収し, 質量増加量から C と H の質量がわかる.

  3. 1. で測り取った質量から, 2. で求めた C と H の質量を引く. これが O の質量

  4. C, H, Oの質量[g] と, C, H, Oのモル質量[g/mol]より, C, H, Oのモル比が調べられる.
