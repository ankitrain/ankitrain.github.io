アルカリ金属

1族の金属

Li, Na, K, Rb, Cs, Fr

語呂: リッチ な 彼女が ルビー せしめて フランスへ

単体

製法

溶融塩電解をする
水溶液を用いた電気分解ではH2Oが反応してH2が出てしまう. → 金属が出てこない

特徴

  1. 酸化されると塩基性酸化物を生じる

    4Na + O2 → 2Na2O
    Ma2O + H2O → 2NaOH

  2. 水や酸素と容易に反応

    そのため石油中で保存が必要

NaOH

特徴

  1. 潮解性がある.

    (空気中にさらすと周りに水がつく)

  2. 酸性酸化物(CO2)と反応

Na2CO3

水和物

室温下で再結晶するとNa2CO3·10H2O(半透明)ができる.

乾燥空気下で放置すると風解して, Na2CO3·H2O(白色)になる.

製法

Na2CO3はアンモニアソーダ法(ソルベー法) でつくる

[A], [B] 2つが同時進行で行われる

[A]

  1. 飽和NaClaqに (α)CO2, (β)NH3を入れる
    ⇒ NaHCO3とNH4Clが発生する

  2. 1. で発生したNaHCO3を熱分解する
    ⇒ Na2CO3と, (α)CO2, H2Oが発生する

[B]

  1. CaCO3を熱分解する
    ⇒ CaO と (α)CO2が発生する

  2. 1. で発生したCaOを水に入れる
    ⇒ Ca(OH)2が発生

  3. 2. で発生したCa(OH)2と, [A]1. で発生したNH4Clを反応させてCaCl2, (β)NH3, H2O を得る

    (弱塩基の遊離)

★ (α), (β) は, 再利用される.

全部の工程をまとめた反応式は,

2NaCl + CaCO3 → CaCl2 + Na2CO3

アルカリ土類金属

2族の金属

Be, Mg, Ca, Sr, Ba, Ra

語呂: Be, Mg, キャッ スルの バ ラ

アルカリ土類金属は, CrO42-塩で, 沈殿が発生する特徴がある

CaCO3

石灰石, 貝殻の主成分であり, チョーク, ガラスの原料でもある.

HClと, 弱酸の遊離をしながら反応する.

CaCO3 + 2HCl → CaCl2 + H2O + CO2

CaO

生石灰ともよばれる.

水と激しく反応して, Ca(OH)2を生じる.

CaO + H2O → Ca(OH)2

製法

CaCO3を熱分解する.

CaCO3 → CaO + CO2

Ca(OH)2

消石灰とも呼ばれ, 水溶液( Ca(OH)2aq )は石灰水と呼ばれる.

製法

CaOと水を反応させる.

石灰水と二酸化炭素

石灰水はしばしば二酸化炭素の検出に用いられる.

  1. Ca(OH)2aq に CO2 を入れると, CaCO3の白色沈殿が発生する.

    Ca(OH)2 + CO2 → CaCO3↓ + H2O

  2. 1. の水溶液に更に CO2 を入れると, Ca(HCO3)2が発生して, 沈殿がなくなり, 水溶液が無色透明になる.

    CaCO3 + H2O + CO2 ⇄ Ca(HCO3)2

CaSO4

天然ではCaSO4·2H2O(セッコウ)として存在する.

それに, 熱を加えると, CaSO2· ½H2O(白色粉末状, 焼きセッコウ)となり, 水を加えると, 元に戻る.

CaCl2

潮解性, 吸湿性が高く, 中性の気体の, 乾燥剤に用いられる.

沈殿

硫酸塩が沈殿する金属

Ba+, Ca2+, Sr2+, Pb2+

語呂: 馬 鹿に する な 硫酸

炭酸塩, シュウ酸塩が沈殿する金属

Ba+, Ca2+, Sr2+

語呂: 馬 鹿に する 炭·シュウ酸

なお, 硝酸塩はどれでも水によく溶ける.

Al - アルミニウム

単体

Alは両性元素である.

三との反応

2Al + 6HCl → 2AlCl3 + 3H2

塩基との反応

2Al + 2NaOH + 6H2O → 2Na[Al(OH)4] +3H2

合金

ジュラルミン
Al + Mg + Cu

AlO3

別名アルミナ

両性酸化物であり, 濃H2SO4aqや濃HNO3と反応すると不動態を形成する.

微量の不純物を含むものにルビー, サファイアがあり, 宝石として用いられる.

人工的にアルミニウムに酸化被膜を付けた製品をアルマイトといい, フライパンとかに使われる.

ミョウバン

化学式
AlK(SO4)2·12H2O

語呂: 歩 く 龍つ いに 水

Al2(SO4)3とK2SO4の混合溶液を濃縮して得られる.

2つ以上の塩からできている塩なので複塩という.

Al3+

塩基と反応して, Al(OH)3(白色沈殿)を生じる.

ただし, さらに過剰にNaOHを入れると, Na[Al(OH)4] (テトラヒドロキシドアルミン酸ナトリウム)ができる.

Al 製法

ボーキサイト(主成分:AlO3· nH2O, 副成分:Fe2O3)からつくる.

  1. ボーキサイトに濃NaOHと熱を加える

    その後ろ過する.

    ⇒ Fe2O3が取り除かれ, (a)Na[Al(OH)4]が発生する.

  2. (a)Na[Al(OH)4] を水で薄める

    ⇒ (b)Na(OH)3↓と, NaOHが発生する.

  3. Na(OH)3に熱を加える.

    ⇒ AlO3ができる.

  4. 3. の工程でできたものを溶融塩電解する.
