O: 酸素

2種類の同素体が存在

O2: 酸素分子

特徴

色 無色
におい 無臭

製法

過酸化水素(H2O2)に、触媒として酸化マンガン(IV)(MnO2)をぼちゃん。

以下反応式
2H2O2 → 2H2O + O2

工業的には、液体空気を分留して作るらしい。

O3: オゾン

特徴

色 淡青色
におい 特異臭

強い酸化作用を示し、有毒である。
漂白作用がある。

湿ったヨウ化カリウムデンプン紙を青紫色に変える。 (酸化作用のため)

製法

2つのやりかたがある

  • 酸素中や空気中で無声放電をして得る
  • 紫外線を照射する

酸化物

酸性酸化物

水と反応して酸を生じたり、塩基と反応して塩を生じたりする。

主に非金属元素の酸化物が該当する。

e.g.) 酸性酸化物を水にぼちゃん

反応例 生成
CO2 CO2 + H2O → H2CO3 炭酸
NO2 3NO2 + H2O → 2HNO3 + NO 硝酸
SO3 SO3 + H2O → H2SO4 硫酸

塩基性酸化物

水と反応して塩基を生じたり、酸と反応して塩を生じたりする。

主に金属元素の酸化物が該当する。

e.g.) 塩基性酸化物を水にぼちゃん

反応例 生成
Na2O Na2O + H2O → 2NaOH 水酸化ナトリウム
CaO CaO + H2O → Ca(OH)2 水酸化カルシウム

両性酸化物

酸とも塩基とも反応するすごいやつ

つまり酸aqにも塩基aqにも溶ける

元素はAl, Zn, Sn, Pb
(語呂合わせ: あ あ すん なり (溶ける) )

e.g.) 酸化アルミニウムを酸aqや塩基aqにぼちゃん

Al2O3 Al2O3 + 6HCl → 2AlCl3 + 3H2O
Al2O3 + 2NaOH + 3H2O → 2Na[Al(OH)4]

S: 硫黄

同素体

斜方硫黄 単斜硫黄 ゴム状硫黄
化学式 S8 S8 Sx
色 黄色 黄色 褐色
形 塊状 針状 ゴム状
CS2溶解 ○ ○ ×

硫黄の性質

硫黄は水には溶けないが、二硫化炭素(CS2)には溶ける。

熱すると、青色の炎を上げて、二酸化硫黄(SO2)を生じる

H2S: 硫化水素

無色で腐卵臭の有毒気体

火山ガスに含まれる

製法

H2S は、硫化鉄(II)(FeS)に塩酸や希硫酸を加えて発生させる。

FeS + H2SO4 → FeSO4 + H2S

性質

  1. H2Sは、水に少し溶ける。

  2. H2S aq は、弱い酸性を示す。 [酸性/塩基性]

    H2S ⇄ H+ + HS-
    HS- ⇄ H+ + S2-

  3. H2Sは、還元作用を示す。

    ヨウ素や二酸化硫黄と反応する。

SO2: 二酸化硫黄

無色で刺激臭の有毒気体。

火山ガスに含まれる。

製法

亜硫酸水素ナトリウム(NaHSO3)や、亜硫酸ナトリウム(Na2SO3)に、希硫酸や塩酸を加えて得る。

2NaHSO3 + H2SO4 → Na2SO4 + 2H2O + 2SO2

銅に、濃硫酸を加えて、加熱しても得られる。

Cu + 2H2SO4 → CuSO4 + 2H2O + SO2

性質

  • 水によく溶ける。

    SO2が溶けている水溶液を亜硫酸という。

  • 酸化還元反応においては、主に還元剤としてはたらく。

H2SO4: 硫酸

製法

接触法という方法で作られる。

以下作り方

  1. 酸化バナジウム(V)(化学式: V2O5)を触媒に用いて、二酸化硫黄(SO2)を酸化して、三酸化硫黄(SO3)にする。

    2SO2 + O2 → 2SO3

  2. 1.でできた物質を濃硫酸に吸収させて、発煙硫酸とする。

  3. 2.でできた物質を希硫酸とまぜまぜして、濃硫酸の完成。

補足: SO3は、H2Oに直接ぼちゃんすると、とてつもなく大きく発熱するから、直接H2Oにぼちゃんしたりはしない。

濃硫酸の性質

  • 市販の濃硫酸は、濃度約98% (約18mol/L)である。

  • 密度が大きい。(1.8 g/cm3)

  • 強い吸湿性 ⇒ 乾燥材として用いられる

  • 有機化合物から、O原子とH原子を、H2Oと同じ組成で奪う。(脱水作用)

    e.g.) C12H22O11 → 12C + 11H2O

  • 加熱した硫酸(熱硫酸)は、強い酸化作用を示す。

    金, 白金以外のほとんどの金属と反応して、二酸化硫黄を発生させる。

    e.g.) Cu + 2H2SO4 → CuSO4 + 2H2O + SO2

希硫酸の性質

濃硫酸を水に溶かすと発熱して希硫酸になる。

希硫酸は、水素よりもイオン化傾向の大きい金属と反応して、H2を発生させる。

N: 窒素

N2: 窒素分子

製法

亜硝酸アンモニウム(NH4NO2)を加熱することで得られる。

NH4NO2 → 2H2O + N2

工業的には、O2と同様に液体空気の分留で得られる。

性質

燃焼させるとNOなどの窒素酸化物を発生させる。

NH3: アンモニア

NH3は、無色、刺激臭のする気体である。

製法

工業的には、ハーバー・ボッシュ法という方法が使われる。

これは、Fe3O4などの触媒を用いて、N2とH2から直接合成する方法である。

N2 + 3H2 ⇄ 2NH3

性質

  • 水によく溶け、塩基性を示す[酸性/塩基性/中性]

  • 蒸発熱が大きいので、 NH3(液) は気体になるために、周囲の熱を奪うため、業務用の冷蔵庫の冷媒などに利用されている。

NO: 一酸化窒素

NOは無色の気体である。

製法

銅と希硝酸を反応させて得られる。

3Cu + 8HNO3(希) → 3Cu(NO3)2 + 4H2O + 2NO

性質

  • O2と容易に反応する
    反応するとNO2を生じる

  • 水にとけにくい[やすい/にくい]

NO2: 二酸化窒素

NO2は、赤褐色の有毒な気体である。

製法

銅と濃硫酸を反応させて得られる。

Cu + 4HNO3(濃) → Cu(NO3)2 + 2H2O +2NO

性質

  • 酸性酸化物である

  • 常温でN2O2と平衡状態である。

HNO3: 硝酸

製法

工業的にはオストワルト法によって作られる。

  1. Ptなどを触媒として、NH3を酸化し、NOにする。

    4NH3 + 5O2 → 4NO + 6H2O

  2. NOを酸化して、NO2にする。

    2NO + O2 → 2NO2

  3. NO2を冷水に吸収させて、HNO3(硝酸)をつくる。
    NOもはっせいするので、これは 2. に再利用する。

    3NO2 + H2O → 2HNO3 + NO

性質

HNO3は、無色で揮発性液体である。

  • 強い酸性を示す。

  • 酸化作用が強い (つまりHNO3は酸化物である)

  • 光や熱で分解されやすい
    ⇒ 褐色容器で保存する。

  • Cu, Ag などの金属と反応し、NO2を生じる。

    しかし、Fe, Al, Ni などとは表面に、緻密な酸化被膜を生じてしまい、これ以上反応が進まなくなってしまう。
    ⇒ 「不働態」

P4O10: 十酸化四リン

製法

Pを燃焼することで得られる。

4P + 5O2 → P4O10

性質

  • 吸湿性がある。
    ⇒ 乾燥剤として利用

  • 潮解性を示す。

  • 酸性酸化物であり、水に溶かして、加熱すると、H3PO4が得られる。

H3PO4: リン酸

製法

P4O10を水に溶かして加熱することによって得られる。

性質

水溶液中では3段階に電離して酸性を示す。

リン酸塩

リン酸カルシウムと硫酸を反応させると, Ca(H2PO4)2(リン酸二水素カルシウム)とCaSO4(硫酸カルシウム)の混合物が得られる。

この混合物を過リン酸石灰といい, リン酸肥料の1つである。

Ca3(PO4)2 + 2H2SO4 → Ca(H2PO4)2 + 2CaSO4

リン酸カルシウムとリン酸を反応させると, Ca(H2PO4)2(リン酸二水素カルシウム)を主に含む物質が得られる。

この物質を重過リン酸石灰といい, リン酸肥料の1つである。

Ca3(PO4)2 + 4H3PO4 → 3Ca(H2PO4)2

C: 炭素

同素体

  • ダイヤモンド
    無色透明, 硬度·光の屈折率が高い
    電気を導かない
    宝石, 研磨剤として用いられる
    共有結合結晶
  • グラフェン
    原子1個分の暑さのシートで, 正六角形を形作りながら形成。
    うすい, かるい, 電気をよく導く
    これが多数積み重なった構造のやつを黒鉛という。
  • フラーレン
    分子式C60やC70で示される物質
    構造は赤ちゃん用のボールのようなもの
    電気を導かない, 色は黒
  • カーボンナノチューブ
    黒煙の層を筒状に丸めた構造
    色は黒, 強度が大きい, 電気を導く
    金額的にはお高いが, 近年は製造技術が発達し, 少しずつ安くなっていっている(けど高い)(無駄情報)

CO: 一酸化炭素

製法

  • 炭素の不完全燃焼でえられる。

    また, 炭素による, 二酸化炭素の還元によっても生じる

    CO2 + C → 2CO

  • 実験室では, ギ酸 (HCOOH)を濃硫酸で脱水して得られる。

    HCOOH → CO + H2O

  • 工業的には, 赤熱したコークスに高温の水蒸気を反応させて作る

    C + H2O → H2 + CO

性質

無色, 無臭の有毒気体。

空気中で, 淡青色の炎を上げて燃焼する。

CO2: 二酸化炭素

製法

実験室では, キップの装置などで, 石灰石に塩酸を加えて得られる

CaCO3 + 2HCl → CaCl2 + H2O +CO2
(弱酸の遊離)

工業的には, 石灰石の熱分解でえられる。

性質

  • 無色無臭。
  • 水に少し溶けて, 溶けた水溶液は, 弱酸性[弱酸性/弱塩基性]をしめす。
    CO2 + H2O ⇄ H+ + HCO3-
  • NaOH, Ca(OH)2などと反応して炭酸塩を生じる
  • 検出には石灰水(化学式:Ca(OH)2)を用いる。
    その水溶液にCO2を加えると, 最初はCaCO3の沈殿を生じて白濁する。
    さらにCO2を加え続けると, 発生した沈殿とも反応して, 白濁消滅する。(この時の水溶液の化学式はCa(HCO3)2である)

Si: ケイ素

Siという元素は天然では単体で存在せず, SiO2として, 地殻中に存在する。

cf. 地殻中に存在する元素の量

O > Si > Al > Fe

Si: ケイ素(単体)

製法

天然に存在するSiO2を, 電気炉で(3000度)に加熱して, Cで還元して得られる

SiO2 + 2C → Si + 2CO

性質

  • ダイヤモンド型の共有結合の結晶。
  • 金属のような灰白色の光沢
  • ダイヤモンドよりは, 結合は弱い

利用

半導体として利用

SiC: 炭化ケイ素

製法

ほぼSi(単体)と同じ

天然に存在するSiO2を, 電気炉で(1800度)に加熱して, Cで還元して得られる

SiO2 + 3C → SiC + 2CO

性質

  • SiCはカーボランダムとも呼ばれる。
  • とても固いため, 工業的には研磨剤として用いられたりする。
  • 基本的には共有結合による決勝
  • ダイヤモンドとSi(単体)の中間のような性質を示す。

SiO2: 二酸化ケイ素

石英(水晶), ケイ砂, ケイ石として, 地殻中から算出される。

性質

  • 共有結合の結晶
    ⇒ かたい, 融点が高い
  • 塩酸などに溶けない

  • フッ化水素酸によく溶ける
    (それゆえ, HF aqは, ガラス瓶では保存できない)

    HF(気)との反応で, SiF4が発生する。

    SiO2 + 4HF(気) → SiF4(気) + 2H2O

    HF aqとの反応で, ヘキサフルオロケイ酸(化学式: H2SiF6aq)が発生する。

    SiO2 + 6HF aq → H2SiF6 aq + 2H2O

塩基との反応

SiO2は酸性酸化物であるので, 塩基と反応することで, 塩ができる。

e.g.) NaOHとの反応

  1. 以下の2式を連立する。
    SiO2 + H2O → H2SiO3
    H2SiO3 + 2NaOH → Na2SiO3 + 2H2O
  2. 連立すると以下の式が得られる。
    SiO2 + 2NaOH → Na2SiO3 + H2O

Na2SiO3: ケイ酸ナトリウム

製法

酸性酸化物であるSiO2に, NaOHを加えて得られる。

SiO2 + 2NaOH → Na2SiO3 + H2O

性質

水ガラス
Na2SiO3に水を加えて加熱したもの。
粘性が大きい液体である。

H2SiO3 (ケイ酸) とシリカゲル

ケイ酸の製法

ケイ酸ナトリウムに水を加えて熱した「水ガラス」に, 塩酸を加えて得られる。

Na2SiO3 + 2HCl → H2SiO3 + 2NaCl
弱酸の遊離

シリカゲルの製法

ケイ酸を熱して, 乾燥させる。

シリカゲルの性質

シリカゲルは多孔質の固体で, 様々な分子を吸着できる
⇒ 乾燥剤として利用。
