私は思ったわけです. 教科書にあるような, ベンゼンとその他の置換基がくっついたような書き方を, Webサイト上では再現できないなと.

ということで, 一置換体は, ⌬ ━ Rのように表現し, 二置換体などは, 画像をどっかから持ってきます.

使いづらいとは思いますが, ご理解願います.

ベンゼン - benzene

概要

  • 化学式は C6H6

性質

  • 特有のにおいを持つ 無色液体 .
  • 水溶性は ない
  • 空気中では多量の すす を出しながら燃焼. (Hに対するCが足りず, 不完全燃焼が起こりやすいため)
  • 化学的に安定. 酸塩基, 還元剤酸化剤などとほとんど反応しない.

置換基の位置

置換基 X と結合している炭素原子を基準として, 隣の位置を o位 (オルト位), さらにその隣の位置(Xから1つ飛ばし)を m位 (メタ位), さらに隣(Xの反対側)を p位 (パラ位) と呼ぶ.

ベンゼンの置換反応

ハロゲン化

ベンゼンに 鉄粉(厳密には FeCl3)を触媒として, Cl2 を作用させると, クロロベンゼン(⌬ ━ Cl) ができる.

⌬ + Cl2 → ⌬ ━ Cl + HCl

なお, 子の生成物は水に 難溶

さらに, Cl2 を反応させると, p-ジクロロベンゼン ができる.

スルホン化

ベンゼンに 濃硫酸 を加えて加熱すると, ベンゼンスルホン酸(⌬ ━ SO3H) を生じる.

この物質は, 無色固体で, 水に 可溶, 液性は 強い酸性 を示す.

⌬ + HO ━ SO3H → ⌬ ━ SO3H + H2O

ニトロ化

ベンゼンに 濃硝酸と濃硫酸の混合物(混酸)を加えて加熱すると ニトロベンゼン(⌬ ━ NO2)ができる.

この物質は 淡黄色液体で, 水に難溶である.

⌬ + HO ━ NO2 → ⌬ ━ NO2 + H2O

トルエンのニトロ化

トルエン(⌬ ━ CH3)は, o, p 指向性があり, 混酸を反応させると, o-ニトロトルエンとp-ニトロトルエンができる.

さらにニトロ化を進めると, 2, 4, 6-トリニトロトルエン(C6H2CH3(NO2)3)(通称: TNT)が生成される.

この物質は, 淡黄色の針状結晶であり, 爆薬として用いられる.

アルキル化

ベンゼンに, AlCl3を触媒として, C2H5Clを反応させると エチルベンゼン(⌬ ━ C2H5)が発生する.

ベンゼンの付加反応

水素

PtやNiを触媒として, H2を反応させると, シクロヘキサン(C6H12)が発生する.

塩素

紫外線照射によって, ベンゼンに塩素が付加して, 1,2,3,4,5,6-ヘキサクロロシクロヘキサン(C6H6Cl6)を生じる.

フェノール類 - phenols

ベンゼン環炭素原子に ━ OH が直接結合した化合物たち

フェノール類の性質

  • 水溶性が ある.
  • 弱酸性を示す.
  • 塩化鉄(III)水溶液を加えると, 青紫色や赤紫色に呈色する.
    ⇒ 検出反応に使われる.
  • アルコールと似た性質を示す.

いろいろなフェノール類

  • ⌬ ━ OH
    名前: フェノール
    詳細は後述
  • C6H4(OH)CH3
    名前: クレゾール
    メチル基と, ヒドロキシ基を持っている.
    消毒液として用いられている.
    o-クレゾール m-クレゾール p-クレゾール
  • C10H7OH
    名前: ナフトール
    (右図は 1-ナフトール)
    染料として用いられている.

フェノール類の反応

酸としての フェノール類の反応

フェノール類はわずかに電離して水素イオンを生じて弱酸性を示す.

⌬ ━ OH ⇄ ⌬ ━ O- + H+

塩基である, 水酸化ナトリウムによって中和されて, ナトリウムフェノキシド(⌬ ━ ONa) を生じる.

⌬ ━ OH + NaOH → ⌬ ━ ONa + H2O

ナトリウムフェノキシドの 弱酸の遊離

フェノール類より強い酸である, 塩酸, 硫酸, カルボン酸(酢酸など), 炭酸などを通じると, フェノールを生じる.

⌬━ONa + CO2 + H2O → ⌬━OH + NaHCO3

⌬━ONa + HCl → ⌬━OH + NaCl

フェノールとナトリウムの反応

アルコールと似た反応

2 ⌬━OH + 2Na → 2 ⌬━ONa + H2

フェノール類のアセチル化

フェノール類のOH基と反応して エステルを形成しやすい

特にフェノールにおいては, 無水酢酸((CH3CO)2O)を作用させることが多い.
この場合, フェノールは, アセチル化して, 酢酸フェニル(⌬━OCOCH3) と 酢酸 を生じる.

⌬━O━H + O(CO━CH3)2 → ⌬━OCOCH3 + CH3COOH

⌬━OH の水素原子の置換反応

ここでは「フェノール類」ではなく, 「フェノール」に起こる反応を見る.

臭素化

フェノールの水溶液に, 臭素水を加えると, 直ちに 2, 4, 6-トリブロモフェノールを生じる.
この物質は 白 色沈殿 として析出する.

ニトロ化

フェノールの水溶液に, 濃硝酸と濃硫酸を加えると, 2, 4, 6-トリニトロフェノール(別名: ピクリン酸) ができる.
生成物には, 爆発性がある.

製法

クメン法

  1. ベンゼン に プロペン を反応させる.

    結果, クメン(C6H5━CH(CH3)2) ができる.

    ⌬ + CH3CH═CH2 → ⌬━CH(CH3)2

    生成物の構造式

  2. 1. でできた クメンを酸化する.

    クメンヒドロペルオキシド(C6H5━C(CH3)2OOH)ができる.

    生成物の構造式

  3. 2. の生成物に H2SO4を通じる.

    主生成物として, フェノール(⌬━OH)が生じる.
    副生成物として, アセトン(CH3━CO━CH3)が生じる.

アルカリ融解による製法

  1. ベンゼンスルホン酸(⌬━SO3H)を, NaOHで中和する.

    ⌬━SO3H + NaOH → ⌬━SO3Na + H2O

  2. 1. の生成物に NaOH を加えて, "アルカリ融解" する.

    ナトリウムフェノキシド(⌬━ONa) が生成する.

  3. フェノールよりも強い酸を用いて, 2.の生成物をフェノールにする. (弱酸の遊離)

クロロベンゼンルート

  1. クロロベンゼン(⌬━Cl) に, 高温·高圧の条件下で, NaOHaqを通じる.

    ナトリウムフェノキシド(⌬━ONa)が生じる.

  2. フェノールよりも強い酸を用いて, 2.の生成物をフェノールにする. (弱酸の遊離)

芳香族カルボン酸 - aromatc carboxylic acid

芳香族カルボン酸の性質

  • 水に溶けにくいけど水に溶ける.
  • 弱酸性を示す. (フェノールより, 強い)
  • NaHCO3aqに, CO2を生じながら溶ける.

安息香酸 - benzoic acid

ベンゼンに COOH基が1つくっついている.

製法

トルエン(⌬━CH3) を KMnO4 で酸化する.

派生(?)

エタノール(C2H5OH)を反応させると, 安息香酸エチル(⌬━COO━C2H5)ができる. (エステル化)

フタル酸 - phthalic acid

ベンゼンに COOH基 が2つくっついている

フタル酸 イソフタル酸 テレフタル酸

無水フタル酸

フタル酸を加熱すると, 分子内脱水され, 無水フタル酸ができる.

構造式

製法

o-キシレン を KMnO4で酸化すると, フタル酸ができる.

m-キシレン を KMnO4で参加すると, イソフタル酸ができる.

p-キシレン を KMnO4で酸化すると, テレフタル酸ができる.

ナフタレン を, V2O5を触媒として, 酸化すると, 無水フタル酸ができる.

PET

テレフタル酸を, エチレングリコール(HO━CH2CH2━OH)と縮合重合すると, ポリマーである ポリエチレンテレフタレート(略記: PET)ができる.

構造式 Wikipediaから引っ張ってきた画像.

OH基とCOOH基を1つずつもつ脂肪族カルボン酸.

サリチル酸 - salicylic acid

フェノールとしての性質と, カルボン酸としての性質両方を示す.

FeCl3aqを通じると 赤紫色 に呈色する.
NaHCO3aqに溶解 する .

医薬品の原料として用いる.

アセチルサリチル酸 - acetylsalicylic acid

別名: アスピリン

サリチル酸 を, 無水酢酸でアセチル化することで, できる.
(この反応では 酢酸 もできる.)

FeCl3で赤紫色に呈色しない.
NaHCO3aqに溶解する.

⇒ 特有の, 芳香臭

FeCl3で赤紫色に呈色する.
NaHCO3aqに溶解しない.

日常では, 消炎鎮痛用湿布薬に用いられている.

アニリン - aniline

ベンゼンに, NH2基が1つくっついている.

水に 難溶. 弱塩基 性を示す.

製法

実験室では, ニトロベンゼンを 濃塩酸とSn or Feで還元し, アニリン塩酸塩(⌬━NH3Cl)を生成し, NaOHaqを加えて(弱塩基の遊離反応) アニリンを生成する.

以下 ニトロベンゼンからアニリン塩酸塩ができる化学反応式

2⌬━NO2 + 3Sn + 14HCl → 2⌬━NH3Cl + 3SnCl4 + 4H2O

工業的には, ニトロベンゼンを, PtやNiを触媒として, 水素を還元して生成する.

反応 (中和と弱塩基の遊離)

アニリンを, HClで中和すると, アニリン塩酸塩(⌬━NH3Cl)ができる.
この物質は, 水に 可溶.

この物質は, NaOHaq を加えると, アニリンが遊離する.

反応 (呈色反応)

アニリンに さらし粉水溶液を加えると酸化され, 赤紫色を呈する.

反応 with 二クロム酸カリウム

硫酸酸性条件下で, K2Cr2O7を加えると, 酸化され, アニリンブラックと呼ばれる物質に変化する.

この物質は水に難溶で, 黒色染料として用いられている.

反応 (アセチル化)

アニリンに 無水酢酸を加えると, アセチル化がおこり, アセトアニリド(⌬━NHCOCH3)ができる.

⌬━NH2 + O(COCH3)2 → ⌬━NHCOCH3 + CH3COOH

アニリンに 酢酸と熱を加えてもアセチル化がおこる.

生成物にある, ━CO━NH━をアミド結合という.
これらは, 酸, 塩基の存在下で, 加水分解される.

アゾ化合物 - azo

━N═N━を アゾ基といい, それを含む化合物をアゾ化合物という.

ジアゾ

━N+≡N の構造を持つジアゾニウム円を生じる反応のこと.

アニリン を 塩酸 に溶かして氷で冷却したうえで, 亜硝酸ナトリウム(NaNO2)の水溶液を加えると, 塩化ベンゼンジアゾニウム(C6H5N2Cl)を生じる.

⌬━NH2 + 2HCl + NaNO2 → ⌬━N+≡NCl- + NaCl + 2H2O

FYI: 冷却しながらやらないと

塩化ベンゼンジアゾニウムは, 低温状態では安定に存在するが, 水温が高くなると水と反応して, フェノールと 窒素を生じてしまう.

カップリング反応

ジアゾニウム塩から, アゾ化合物(━N═N━基を持つ化合物)を作る反応

塩化ベンゼンジアゾニウム(⌬━N2Cl)の水溶液にフェノールを混ぜた水酸化ナトリウム水溶液を加えて, 橙色のp-ヒドロキシアゾベンゼンを生じる反応が有名.

⌬━N2Cl + ⌬━ONa → ⌬━N═N━⌬━OH + NaCl
